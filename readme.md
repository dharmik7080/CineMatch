## 📖 About

CineMatch is a premium movie discovery platform inspired by modern streaming services. It allows users to discover trending movies and TV shows, search an extensive library, build a personalized watchlist, and enjoy a clean cinematic browsing experience powered by the TMDB API.

---

## ✨ Features

- 🎥 Browse Trending Movies
- 🔍 Search Movies & TV Shows
- ❤️ Personal Watchlist
- 📊 Analytics Dashboard
- 🌙 Dark Mode UI
- 📱 Fully Responsive
- ⚡ Fast & Smooth Experience

---

## 🛠 Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **API:** TMDB API
- **Authentication:** Django Authentication
